# P1-Lautaro-Pereyra
Lograr que al tocar un pulsador dos servomotores avancen en intervalos 10°.
Lograr que al tocar otro pulsador los dos servomotores retrocedan en intervalos 10°.
Tiene que haber una función avanzar y una función retroceder.
Si el pulsador deja de ser presionado dejan de moverse los servos.
Entrega a través de su repositorio de GitHub de Proyecto Informático, En una branch con el nombre del TP, Tiene que estar el código en formato .ino, una instantánea del esquema y la consigna pegada en el archivo README.
